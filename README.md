# CODSOFT-2
## Job Board Website

A modern Job Board web application built using React.js that allows employers to post job openings and candidates to browse and apply for jobs. The project focuses on clean UI, smooth user interactions, and modular component-based design.

## Features

### -Home Page
###### Welcome message
###### Simple and clean landing layout
###### Job Listings
###### View multiple job openings
###### Clear separation between job cards
###### Easy navigation to job details

### -Job Details Page
###### Detailed job description
###### Company, location, experience, and skills

### -Employer Dashboard
###### Overview of employer actions
###### Job posting and application management (UI level)

### -Candidate Dashboard
###### Profile management section
###### Update profile interaction

### -UI & UX Enhancements
###### Dark navbar with logo-styled title
###### Custom fonts and hover effects
###### Animated buttons and feedback indicators
###### Responsive layout (works on different screen sizes)

### -Tech Stack

#### -Frontend
###### React.js
###### React Router DOM
###### HTML5
###### CSS3

#### -Backend 
###### Node.js
###### Express.js
###### MongoDB / PostgreSQL
